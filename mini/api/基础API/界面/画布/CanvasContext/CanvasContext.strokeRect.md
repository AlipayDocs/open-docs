# 简介

**CanvasContext.strokeRect** 用于画非填充矩形。

## 使用限制

- 用 `setFillStroke()` 设置矩形线条的颜色，如果没设置默认是 `black`。
- 此 API 支持个人支付宝小程序、企业支付宝小程序使用。

# 接口调用

## 示例代码

### .js 示例代码

```javascript
//.js
const ctx = my.createCanvasContext('canvas');
ctx.setStrokeStyle('blue');
ctx.strokeRect(20, 20, 250, 80);
ctx.draw();
```

## 入参

**number x**

矩形左上角的 x 坐标。

**number y**

矩形左上角的 y 坐标。

**number width**

矩形路径宽度。

**number height**

矩形路径高度。
