
## Q：小程序右上角的 分享与收藏 可以设置颜色吗？
A：这是默认的，无法设置颜色。

## Q：小程序胶囊按钮内的菜单页是否可以支持自定义修改？

## Q：导航栏的字体颜色可以自定义修改吗？ 
A：导航栏字体颜色无法自定义修改，但是可以通过修改背景颜色，自动调整变成黑色或白色的导航栏字体颜色。
