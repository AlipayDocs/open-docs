
# 简介
真机预览可采用 “云构建” 与 “本地构建” 两种方式，命令行跟界面都可使用。

# 命令行
```shell
miniu mini preview -i appId -p project # 云构建真机
miniu mini preview --local-build -i appId -p project # 本地构建
```
