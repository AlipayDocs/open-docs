## 属性说明

| **属性** | **类型** | **描述** |
| --- | --- | --- |
| value | String | 组件值，选中时 change 事件会携带的 value。 |
| checked | Boolean | 当前是否选中，可用来设置默认选中。<br />**可选值：** true、false。<br />**默认值：** false |
| disabled | Boolean | 是否禁用。<br />**可选值：** true、false。<br />**默认值：** false |
| id | String | 与 label 组件的 for 属性组合使用。 |
