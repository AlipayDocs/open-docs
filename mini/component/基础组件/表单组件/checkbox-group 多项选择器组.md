# 简介

多项选择器组，内部由多个 [checkbox](https://opendocs.alipay.com/mini/component/checkbox) 组成。

# 使用

## 在线示例

## 属性说明

| 属性 | 类型 | 描述 |
| --- | --- | --- |
| name | String | 组件名字，用于表单提交获取数据。 |
| onChange | EventHandle | 选中项发生改变时触发，`detail = {value: 选中的 checkbox 项 value 的值}`。 |
