## 属性说明
| **属性** | **类型** | **描述** |
| --- | --- | --- |
| name | String | 组件名字，用于表单提交获取数据。 |
| onChange | EventHandle | 中选中项发生改变时触发， `detail = {value: 选中的 checkbox 项 value 的值}`。  |
